-  模拟器开启
    1：查看本机支持的模拟器

        flutter emulator
        
    2：启动模拟器(launch后面接模拟器名字)

        flutter emulators --launch apple_ios_simulator

    3：查看可用的设备

        flutter devices

    4：将应用运行到模拟器上(-d 是选择模拟器运行，如果只有一个模拟器可以直接run)
    
        flutter run -d C91AD035-B863-4D6B-9C30-C8CCAE3567DC
